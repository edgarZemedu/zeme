## 🚀 Instrucciones de instalación

### 1. Copia el archivo `index.html` a la raíz de tu repositorio.

### 2. Crea una carpeta llamada `assets/` y copia dentro todas las imágenes que te he adjuntado.

### 3. Sube todo a tu repositorio:
```bash
git add .
git commit -m "feat: add Ethio-Spain Runway website"
git push origin main
```

### 4. Activa GitHub Pages (si usas GitHub):
- Ve a **Settings → Pages**
- Source: **Deploy from a branch → main → / (root)**
- Tu web estará en: `https://TU_USUARIO.github.io/TU_REPO/`
