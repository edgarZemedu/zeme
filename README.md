# ethiospainrunway

Este es un repositorio para mi primera página web estática.

## Estructura del proyecto

- `index.html` – Archivo principal con el contenido.
- `styles.css` – Hojas de estilo para la página.

## Despliegue en GitHub Pages

1. Asegúrate de que tu rama `main` esté actualizada y sube estos archivos:
   ```sh
   git add index.html styles.css README.md
   git commit -m "Añadir primer sitio estático"
   git push origin main
   ```
2. Ve a la configuración del repositorio en GitHub.
3. En «Pages», selecciona la rama `main` y la carpeta `/ (root)` como fuente.
4. Guarda la configuración. En unos minutos tu sitio estará disponible en `https://<usuario>.github.io/zeme`.

¡Listo! Ahora tienes una web estática básica desplegada con GitHub Pages. Puedes editar `index.html` y `styles.css` para personalizarla.
